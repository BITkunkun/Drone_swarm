主流编队控制方法：
（1）领航跟随法（Leader-Follower）
原理‌：指定一架无人机为领航者，其余跟随者基于领航者状态调整自身位置。
‌优点‌：逻辑简单，计算量低，适合线性编队。
‌缺点‌：过度依赖领航者，难以处理复杂队形变换。
